<div className="section">

<h2 className="sectionHeading">Architecture Highlights</h2>

<div className="gridList">
  <div className="gridItem">Google Kubernetes Engine (GKE) multi-cluster setup</div>
  <div className="gridItem">Sidecar container architecture for app updates</div>
  <div className="gridItem">OpenResty (Nginx) for reverse proxy and auto SSL</div>
  <div className="gridItem">MySQL database with persistent storage</div>
  <div className="gridItem">phpMyAdmin for database management</div>
  <div className="gridItem">BigBlueButton integration for e-learning</div>
</div>

</div>

---

<div className="section">

<h2 className="sectionHeading">Platform Architecture</h2>

<div className="gridList">
  <div className="gridItem">Pods hosting multiple containers (Nginx, PHP-FPM, App)</div>
  <div className="gridItem">Kubernetes Services exposing applications</div>
  <div className="gridItem">Persistent Volumes and Claims for storage</div>
  <div className="gridItem">Private container registry (GCR)</div>
</div>

</div>

---

<div className="section">

<h2 className="sectionHeading">CI/CD Pipeline</h2>

<div className="gridList">
  <div className="gridItem">Jenkins pipelines triggered via Git branches</div>
  <div className="gridItem">Dev: Direct code sync without image rebuild</div>
  <div className="gridItem">Stage: Image build and deployment via Helm</div>
  <div className="gridItem">Prod: Controlled manual deployment process</div>
</div>

</div>
